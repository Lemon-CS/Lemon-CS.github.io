CSS入门学习笔记

CSS

1.1 什么是CSS？

css是指层叠样式表，样式定义如何显示html元素

语法

css主要用于设置HTML页面中的文本内容（）字体、大小、对齐方式等）、图形的外形（宽高、边框样式、边距等）以及版面的布局等外观显示样。

1.2 引入css样式表

1.内部样式表（内嵌式）

内嵌式是将css代码集中写在HTML代码的头部head标签中，并用style标签定义。

2.行内式（内联样式）

是通过标签的style属性来设置元素的样式

<标签名 style=“属性1：属性值1；属性2：属性值2”> 内容</标签名>

3.外部样式表（外链式）

嵌入式是将所有的样式放在一个或多个以.css为扩展名的外部样式表文件中，通过标签将外部样式表文件链接到HTML文档中。

1.3 CSS的选择器

  1. 元素选择器（标签选择器）

    指用HTML元素名（或标签名）作为选择器，按标签名分类，为页面中某一类标签指定统一的CSS样式

    格式如下： 标签名（元素名）{属性1：属性值1；属性2：属性值2；}

  1. 类选择器

    类选择器允许以一种独立于文档元素的方式来指定样式。

    该选择器可以单独使用，也可以与其他元素结合使用。

    格式：.类名{属性1：属性值1；属性2：属性值2}

    调用时<标签名 class=“类名>即可

  1. id选择器

    id选择器使用”#“进行标识，格式如下：

    #id名{属性1：属性值1；属性2：属性值2}

    id名即为HTML元素的id属性值，元素的id是唯一的。

    类选择器可以多次重复使用，而id选择器是唯一的。

  2. 通配符选择器

    通配符选择器用”*“号表示，是所有选择器中范围最广的，能匹配页面中所有的元素。

    *{属性1:属性值1；属性2：属性值2；}
